How to Embed a YouTube Video

Step 1: Find a video on YouTube

Go to YouTube and find the video you want to embed on your page.

Step 2: Click the "Share" button

Below the video, click the "Share" button to open sharing options.

Step 3: Click "Embed" and copy the iframe code

In the sharing options, click "Embed" and then copy the iframe code provided.

Step 4: Paste the code into your HTML file

Paste the copied iframe code into your HTML file where you want the video to appear.
