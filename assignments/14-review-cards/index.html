What does CSS stand for?

Cascading Style Sheets

What tag creates a paragraph?

The <p> tag

What property changes text color?

The color property

What does the <a> tag do?

Creates a hyperlink

What is the box model?

Content, padding, border, and margin

How do you make text bold in HTML?

Use the <b> or <strong> tag

What does float do?

Positions an element to the left or right

What is a CSS pseudo class?

A keyword added to a selector for a special state
